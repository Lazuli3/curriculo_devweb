Dicionário de Elementos HTML

Referência de tags utilizadas nas páginas

Esta página apresenta um dicionário com todas as tags HTML utilizadas nas páginas index.html e html_dict.html. Para cada tag, é descrito o seu propósito e função dentro do documento.

TAGS UTILIZADAS EM index.html

Tag Descrição
<!DOCTYPE html> Declaração que informa ao navegador que o documento é do tipo HTML5.
<html> Elemento raiz que envolve todo o conteúdo da página HTML. O atributo lang define o idioma do documento.
<head> Contém metadados e informações sobre o documento que não são exibidos diretamente na página, como título, links de estilo e scripts.
<meta> Define metadados do documento. Utilizado para configurar o charset (conjunto de caracteres) e o viewport (comportamento em telas móveis).
<title> Define o título da aba/janela do navegador e é usado por mecanismos de busca.
<link> Vincula recursos externos ao documento, como folhas de estilo CSS e ícones (favicon).
<script> Incorpora ou referencia scripts JavaScript. Aqui usado para carregar os ícones do Font Awesome.
<body> Contém todo o conteúdo visível da página, como textos, imagens e outros elementos.
<div> Elemento de bloco genérico usado para agrupar e organizar outros elementos para fins de layout e estilização.
<nav> Define uma seção de navegação com links para outras partes da página ou para outras páginas.
<a> Cria hiperlinks para outras páginas, seções da mesma página ou endereços externos. O atributo href define o destino do link.
<header> Representa o cabeçalho de uma seção ou da página, geralmente contendo título, logo e informações introdutórias.
<h1> Título de nível 1 — o mais importante da página. Usado para o nome principal (Pietra Mota).
<p> Define um parágrafo de texto. É um elemento de bloco que adiciona espaçamento automático ao redor do texto.
<img> Exibe uma imagem na página. Os atributos src (caminho) e alt (texto alternativo) são essenciais.
<section> Define uma seção temática do conteúdo, geralmente com um título próprio. Usado para organizar objetivos, educação, experiência, etc.
<h2> Título de nível 2 — subseção dentro da página. Usado para os títulos de cada seção do currículo.
<main> Define o conteúdo principal e único da página, excluindo cabeçalhos, rodapés e navegação.
<ul> Cria uma lista não ordenada (sem numeração). Os itens são definidos pela tag <li>.
<li> Define um item dentro de uma lista ordenada (<ol>) ou não ordenada (<ul>).
<hr> Insere uma linha horizontal de separação visual entre seções do conteúdo.
<i> Originalmente usada para texto em itálico. Aqui utilizada para inserir ícones do Font Awesome através de classes CSS.
<br> Insere uma quebra de linha simples dentro de um bloco de texto, sem criar um novo parágrafo.

TAGS UTILIZADAS EM html_dict.html

Tag Descrição
<!DOCTYPE html> Declaração que informa ao navegador que o documento é do tipo HTML5.
<html> Elemento raiz que envolve todo o conteúdo da página HTML.
<head> Contém metadados da página como título, charset, viewport e estilos embutidos.
<meta> Define metadados do documento. Usado para charset e configuração responsiva do viewport.
<title> Define o título exibido na aba do navegador.
<link> Vincula o favicon (ícone da aba) ao documento.
<script> Carrega o script do Font Awesome para uso de ícones na página.
<style> Permite escrever CSS diretamente dentro do HTML, no elemento <head>. Usado para estilizar toda esta página.
<body> Contém todo o conteúdo visível da página.
<div> Elemento de bloco genérico para agrupamento e layout de conteúdo.
<nav> Define a barra de navegação com links para o currículo e para as seções desta página.
<a> Cria hiperlinks. Usado para voltar ao currículo e navegar entre as seções da página.
<header> Cabeçalho da página com logo e título do dicionário.
<img> Exibe o logo no cabeçalho da página.
<h1> Título principal da página do dicionário.
<h2> Títulos de seção para separar o dicionário do index.html e do html_dict.html.
<hr> Linha separadora entre o cabeçalho e o conteúdo principal.
<main> Contém o conteúdo principal da página — o texto introdutório e as tabelas do dicionário.
<p> Parágrafo de texto introdutório explicando o propósito da página.
<strong> Deixa o texto em negrito com ênfase semântica, indicando importância ao navegador e leitores de tela.
<table> Define uma tabela para organizar os dados do dicionário em linhas e colunas.
<thead> Agrupa as linhas do cabeçalho da tabela (Tag e Descrição).
<tbody> Agrupa as linhas de dados do corpo da tabela.
<tr> Define uma linha dentro de uma tabela.
<th> Define uma célula de cabeçalho em uma tabela. O texto é negrito e centralizado por padrão.
<td> Define uma célula de dados em uma tabela.
<code> Exibe texto em fonte monoespaçada, indicando que o conteúdo é um trecho de código.
<i> Usada para inserir ícones do Font Awesome (seta de voltar no menu de navegação).
